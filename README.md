# 如何使用

-   您可以应用 sLocalStorage 在 [40code](https://www.40code.com/) 社区上。如果您需要在其他社区或本地环境下使用，如果未经过修改，项目存储功能将无法使用。
-   您可以直接按照所述使用，本拓展的逻辑十分简单，只是将您的内容通过 JavaScript 脚本存入浏览器本地存储而已。
